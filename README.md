# Описание проекта
Данный проект представляет собой простую модель системы управления товарами и категориями с использованием объектно-ориентированного программирования

## Функциональность:
**Product**  
Базовый класс для описания товара с основными атрибутами:  

- name — название товара  
- description — описание  
- price — цена (инкапсулированный атрибут с геттером и сеттером, где проверяется положительность цены)  
- quantity — количество на складе  

Реализованы методы:  

- создание товара из словаря (`new_product`)  
- строковое представление (`__str__`, `__repr__`)  
- перегрузка оператора сложения, чтобы складывать общую стоимость двух товаров одного типа (`__add__`)

**Smartphone (наследник Product)**  
Расширяет класс Product дополнительными характеристиками для смартфонов: производительность, модель, объем памяти, цвет.

**LawnGrass (наследник Product)**  
Определяет товар типа газонной травы с дополнительными атрибутами: страна происхождения, срок прорастания, цвет.

**Category**  
Класс для группировки товаров в категории. Содержит:  

- название и описание категории  
- приватный список товаров, поддерживающий добавление только объектов класса Product или его наследников  
- счетчики общего количества категорий и товаров (статические переменные)  
- методы для получения списка товаров в виде строк  
- безопасный доступ к товарам (через возвращение копии списка)  
- строковое представление, отображающее название категории и суммарное количество товаров в ней
