Проект E-commerce
Работа по освоению использования классов и объектов на основе популярной темы e-commerce.

## Использование:
- создана функция , целью которой является предоставление удобного способа хранения и обработки данных о товарах и их классификациях.
- Класс Product предназначен для создания отдельных записей о товарах. Каждая запись включает наименование товара (name), подробное описание (description), цену (price) и количество на складе (quantity). Это позволяет хранить структурированную информацию о конкретных товарах.
- Класс Category служит для группировки товаров по категориям. Категория имеет своё собственное название (name), описание (description) и список товаров (products), относящихся к данной категории. Дополнительно реализованы два глобальных счётчика:
  - category_count отслеживает количество созданных категорий.
  - product_count фиксирует суммарное количество товаров среди всех категорий.
